# Projects

- [Harvest](https://github.com/beshubh/harvest) — A search engine with a concurrent crawler, positional inverted index, SPIMI indexing, k-way merge, and phrase queries.
- [Kafka in Rust](https://github.com/beshubh/codecrafters-kafka-rust) — A from-scratch implementation of the Kafka protocol and log-broker fundamentals.
- [rslox](https://github.com/beshubh/rslox) — A Lox interpreter in Rust, built to learn scanning, parsing, evaluation, and language implementation.
- [Distributed systems in Rust](https://github.com/beshubh/distsys-rust) — Implementations exploring Raft, fault-tolerant key-value storage, transactions, and linearizability.
- [go-db](https://github.com/beshubh/go-db) — A small database in Go for learning storage-engine internals.
- [rust-git](https://github.com/beshubh/rust-git) — A compact Git implementation in Rust.
- [LeetCode GitHub Sync](https://github.com/beshubh/leetcode-gh-sync) — A Firefox extension with a Rust/Wasm synchronization planner.
